Concept Of WWW

  • It Stands For World Wide Web.
  • It is basically a system of internet servers that support specially formatted document.
  • It is a collection of text pages,video,digital photographs ans animations we can access over the internet.
  • User access the World Wide Web facilities via client a web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol.
  2. HTTP is based on the cilent-server architecture model and a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
